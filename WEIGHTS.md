# 模型权重说明

本仓库不提交模型权重文件。请在本地下载或复制权重后放入以下目录：

```text
models/weights/
```

默认文件名：

```text
models/weights/aide_night_best.pth
models/weights/resnet50_ai_detector.pth
models/weights/efficientnet_b0_ai_detector.pth
```

如果文件名不同，可以在系统设置或模型管理页面中选择对应权重。自定义上传的权重默认可放入：

```text
models/weights/custom_uploads/
```

建议将大模型权重放在 GitHub Releases、Hugging Face、网盘或私有对象存储中，不要直接提交到 Git 仓库。
